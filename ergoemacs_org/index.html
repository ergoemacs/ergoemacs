ErgoEmacs

ErgoEmacs is a fantastic text editor.

You download it, and use it for your day job and be productive in the first day.

ErgoEmacs is as easy to use as Notepad, If you are familiar with Microsoft Notepad, you can be productive with it the minute you download it. It is as easy to use as Notepad, Notepad++, BBEdit, Textmate, and easier than IDEs such as Visual Studio, XCode, Eclispe, NetBeans.

ErgoEmacs is a modernized version of emacs. It improves emacs's difficult keyboard shortcuts, menus, outdated terminologies, and with many more added packages.

Features

For a list of features, see: ErgoEmacs Features.

Download

For Windows: ErgoEmacs 1.7 Setup.exe.

For Mac OS X and Linux, you need a emacs 23 installation, then install the ErgoEmacs Elisp Package that will make your emacs behave as ErgoEmacs. Download here: ergoemacs_1_7_1.zip.

See also: Version 1.7 Release Notes.
